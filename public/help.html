Lich Help

Instructions

Key Function
ESC Toggles main menu window
W or ↑ Jump or climb
A or ← Walk left
D or → Walk right
S or ↓ Climb down
I Toggle minimize inventory
C Toggle crafting window
M Toggle map
Left Mouse Button Select item/spell
Left Mouse Button Fire selected spell
Left Mouse Button Place item
Left Mouse Button + SHIFT Place item as background (surface)
Left Mouse Button + SHIFT Place item as flipped variant (object)
Right Mouse Button Interact with an object or open workstation
